HDDB-TABLE-FIELD-EXISTS
#######################

Query existence of field within table
-------------------------------------

:Manual section: 1
:Date: 2017-05-30
:Author: Jan MatÄ›jka yac@blesmrt.net
:Manual group: HDDB manual

SYNOPSIS
========

  hddb table <table> field <field> exists

DESCRIPTION
===========

Exits on 0 if field ``field`` exists within table ``table``. >0 otherwise.
